# BazyD

## Aplikacja na potrzeby projektu 

### Funkcje
- Logowanie
- Odczyt z pliku .db
- Zapis do pliku .db
- Dodanie rekordów do bazy
- Aktualizacja istniejących rekordów w tabelach
- Otwieranie plików PDF
  

### Technologia

- Python Tkinter
- Visual Studio Code

#### Uruchomienie: 
- home i report -> python app.py
- pozostałe pliki -> python (nazwa pliku)
